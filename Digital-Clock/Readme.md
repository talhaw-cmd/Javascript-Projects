# JavaScript Clock Project
About the Project

This is a real-time digital clock built using HTML, CSS, and JavaScript.
The clock displays:

 - Current hours, minutes, and seconds
 - Current day of the week
 - Current date and month

It updates every second using JavaScript and is styled with a modern gradient background and glowing effects.

## Features

 - Real-time hours, minutes, and seconds
 - Displays day of the week, date, and month
 - 12-hour format with automatic updates
 - Modern design with gradient background and blurred glow circles
 - Fully responsive and simple to customize

## Technologies Used

 - **HTML5** – For the structure of the clock
 - **CSS3** – For styling, layout, and glowing effects
 - **JavaScript (ES6)** – For real-time updates and DOM manipulation
