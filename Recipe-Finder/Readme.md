# Recipe Finder Project
About the Project

Recipe Finder is a web application built using HTML, CSS, and JavaScript that allows users to search for recipes from TheMealDB API.
Users can:

 - Search for recipes by name
 - View recipe images, category, and origin
 - Open full recipe instructions in a modal
 - Access YouTube videos and detailed sources for recipes
 - The app is responsive and includes a modern, clean UI design.

## Features

 - Search recipes using keywords
 - Display recipe image, name, category, and area
 - Open recipe instructions in a modal popup
 - Access YouTube video link and detailed recipe source
 - Responsive design for mobile and desktop screens

## Technologies Used

 - **HTML5** – Structure of the app
 - **CSS3** – Styling, layout, and responsiveness
 - **JavaScript (ES6)** – Fetching API data and DOM manipulation

Bootstrap Icons – For close icon in modal
